# Advent of Code

Exercising with Advent of Code: https://adventofcode.com/

## Usage

```
$ lein repl
=> (use 'aoc2019.d1 :reload)
=> (require '[clojure.test :as t])
=> (t/test-ns 'aoc2019.d1)
```
